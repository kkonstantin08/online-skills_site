# Online Skills

Данный проект представляет собой статический сайт, созданный с использованием HTML и CSS.  
Сайт оформлен как лендинг образовательной тематики и сейчас уже включает базовую мобильную адаптацию и якорную навигацию с плавной прокруткой.

#### Структура сайта
1. **Главная секция**: навигационное меню, hero-блок и кнопка "Get Started".
2. **Learn section**: блок с акцентом на ценность обучения и карточками-иконками.
3. **Design & Business sections**: два контентных блока с текстом и декоративными frame-элементами.
4. **About section**: блок "Join New Skills Online With Top Educators".
5. **Footer / Contact**: контакты, партнерские ссылки и дополнительная информация.

#### Технологии
- HTML
- CSS

#### Особенности
- адаптивная верстка для популярных мобильных ширин;
- якорная навигация по секциям;
- плавная прокрутка при переходе по пунктам меню;
- без дополнительных библиотек и сборщиков.

#### Запуск
Проект не требует сборки.  
Достаточно открыть [index.html](./index.html) в браузере или запустить через Live Server в редакторе.

This project is a static HTML and CSS website.

#### Technologies
- HTML
- CSS

#### Features
- responsive layout for common mobile widths;
- anchor navigation between sections;
- smooth scrolling for menu links;
